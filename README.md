## How to run the code

Run backend:

1. cd into reiseutforsker
2. cd into backend
3. run 'npm i'
4. run 'npm start'

Run frontend:

1. cd into reiseutforsker
2. cd into TravelPlanner
3. run 'npm i'
4. run 'npm run dev'
